# Code of Conduct

Be respectful. Disagreement is fine; personal attacks, harassment, and bad-faith behavior are not.

This project channels historical thinkers honestly — including views that may be outdated or uncomfortable. Debate the content; don't harass the people.

Report concerns to the maintainer via GitHub Issues or by contacting the repo owner.

Violations may result in a warning or a ban from contributing, at the maintainer's discretion.
