# Contributing

Thanks for helping grow the roster — or improve the skill itself.

## Ways to help

1. **Add a thinker** — the most common contribution
2. **Improve an existing entry** — clearer method, better works, tighter voice
3. **Improve the skill** — `aristotle/SKILL.md` workflow, guardrails, or docs

## Adding a thinker

Edit [`aristotle/references/thinkers.md`](./aristotle/references/thinkers.md).

**Rules:**
- Deceased thinkers only (living people can speak for themselves)
- Must have a real published body of work
- Use the template at the top of that file — all four layers, kept tight (~10 lines)
- No fabricated quotes or invented works

**Four layers (in priority order):**
1. Core works
2. Method of thought
3. Core commitments
4. Signature moves & voice

Put the new entry under the best-fit domain heading, or add a heading if needed.

## Pull requests

1. Fork and branch from `main`
2. Make your change in the `aristotle/` source files
3. Open a PR with a short description of who/what and why
4. Maintainers will rebuild `aristotle.skill` (it's a zip of `aristotle/`) for releases

## Conduct

Be kind. See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).
